MapleStory Mini Toolkit – Site Plan

W05

Site Name

MapleStory Mini Toolkit – a lightweight, fast set of web tools for MapleStory (KMS) players. The name communicates a focused, utility-first helper that feels quick and portable.

Optional domain: kms-mini-toolkit.org

Site Purpose

  • Provide calculators for stats, potential/additional potential, Star Force cost, and upgrade efficiency.
  • Offer a searchable item database with filters, sorting, pagination, and expandable details.
  • Save favorites and settings in the browser for quick return visits.
  • Publish concise beginner guides and a glossary for new or returning players.

Target Scenarios

  • “If I cube this emblem, how much main stat will I gain on average compared to Star Force?”
  • “Show only level 140–160 warrior gear with Epic+ potential and sort by attack.”
  • “Compare two rings side-by-side and highlight the more efficient upgrade path.”

JavaScript Requirements

  • Objects/arrays with map, filter, reduce, sort.
  • DOM interaction with event listeners and conditional branching.
  • Template literals only for UI strings.
  • localStorage for favorites, recent searches/comparisons, dark mode.
  • Lazy loading images, skeleton cards, focus management, ARIA labels.
