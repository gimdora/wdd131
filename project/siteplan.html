MapleStory Mini Toolkit — Site Plan

Site Name

MapleStory Mini Toolkit — lightweight calculators and item helpers for KMS players.

optional domain kms-mini.tools

Site Purpose

The site provides quick, mobile-first utilities: stat/potential calculators, Star Force cost estimates, upgrade comparison, a searchable item DB, and short guides. It aims to be fast, accessible, and usable on phones during gameplay.

Scenarios

  • How many meso and success chances do I need to take an item from 12★ to 17★?
  • Which of two items gives more final stats after potentials and flames?
  • Where can I look up a weapon and filter by job, level, and rarity?
  • How do cubes and potentials work for beginners?

Color Schema

Brand#0B5CAB
Accent#E94E1B
Background#0A0F1A
Body Text#1F2937
Card#FFFFFF

Brand is used for headings and buttons, Accent for badges and links on hover, Background for the app shell, Body Text for dark text on light cards, and Card for surfaces.

Wireframes

Planned Pages

  • Home — intro, featured tools, recent favorites.
  • Item DB — searchable cards with filters and sorting.
  • Tools — stat/potential calculators, Star Force cost, upgrade comparison.
  • Guides — brief beginner guides and glossary.
  • References — citations page for any external sources.

Core Tech

  • Mobile-first responsive layout using CSS Grid/Flex.
  • Accessible components with WCAG AA color contrast.
  • JavaScript: objects/arrays with array methods, DOM updates, events, conditionals, template literals only, and localStorage for favorites and settings.
  • Images optimized and lazy-loaded.
